# Healthcare-Patient-Analytics-Dashboard
Interactive Healthcare Patient Analytics Dashboard built in Excel using Data Cleaning, IF Functions Pivot Tables, Pivot Charts, KPIs, and Slicers. Analyzed 54,966 patient records to generate insights on medical conditions, admissions, insurance providers, age groups, billing, and medications.
# Healthcare Patient Analytics Dashboard (https://github.com/aamit428-hue/Healthcare-Patient-Analytics-Dashboard/blob/main/DASHBOARD.png)

## Project Overview

This project presents an interactive Healthcare Patient Analytics Dashboard developed in Microsoft Excel using healthcare patient data. The objective was to transform raw healthcare data into meaningful insights through data cleaning, analysis, and visualization techniques.

## Key Features

* Data Cleaning and Preparation
* KPI Creation and Performance Metrics
* Pivot Tables and Pivot Charts
* Interactive Slicers and Filters
* Dynamic Dashboard Design
* Healthcare Data Analysis

## Dashboard Metrics

* Total Patients: 54,966
* Average Billing Amount: 25,544
* Average Length of Stay: 15 Days

## Analysis Performed

* Medical Condition Analysis (Diabetes, Asthma, Hypertension, Arthritis, Cancer, and Obesity)
* Admission Type Analysis (Emergency, Elective, and Urgent)
* Age Group Distribution Analysis
* Insurance Provider Analysis
* Medication Usage Analysis
* Patient Demographics Analysis

## Excel Skills Demonstrated

* Data Cleaning
* IF Functions
* VLOOKUP/XLOOKUP
* Conditional Formatting
* Pivot Tables
* Pivot Charts
* Slicers
* Dashboard Development
* Data Visualization
* Healthcare Analytics

## Key Insights

* Patients aged 60+ represent the largest patient segment.
* Emergency admissions account for a significant portion of hospital visits.
* Billing patterns remain relatively consistent across medical conditions.
* Insurance provider analysis helps identify patient coverage trends.
* Medication analysis highlights common post-discharge prescriptions.
